# Resumo da Atividade: Classes e Objetos #
A atividade consiste em criar duas classes em Java, Médico e Hospital, para praticar o uso de classes e objetos.

A classe Médico possui atributos como CRM, nome e salário. Essa classe inclui métodos para exibir seus dados.

Em seguida, a classe Hospital, gerencia uma lista de objetos Médico. Ela tem métodos para adicionar, remover, e exibir os médicos, seja por um ano de contratação específico ou para imprimir a lista completa.

O objetivo é modelar a relação entre médicos e um hospital usando os conceitos de Programação Orientada a Objetos.
